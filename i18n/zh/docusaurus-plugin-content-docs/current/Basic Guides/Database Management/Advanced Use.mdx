Tier0 支持使用与其深度集成的开源产品，并赋予全部操作权限。
1. 登录 Tier0 平台，选择**工具集** > **高阶使用**。

<img width={650} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/181-1.png" />

2. 单击任一开源产品，即可进入其管理页面。

<img width={650} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/182-1.png" />
