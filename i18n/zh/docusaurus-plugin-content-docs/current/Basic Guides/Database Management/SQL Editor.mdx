你可以对平台内置的数据库执行 SQL 查询操作。

1. 登录平台后，选择 **工具集** > **SQL 编辑器**。

2. 在左侧选择对应的主机和数据库，双击数据表即可查看其内容。

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/74-1.png" />

3. 单击已打开数据表旁的 <img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/75.png" />，开始执行 SQL 查询。

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/76-1.png" />
