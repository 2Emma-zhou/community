:::info
在**数据连接**中处理文件前，需将文件放入对应的容器中。
```js
scp C:\Users\you\Desktop\myfile.txt username@server IP:/home/username/  
 //将文件从本地复制到服务器
docker cp /root/config.json <容器名称或ID>:/app/config.json  
 //将文件从服务器复制到容器
```
:::

## 连接文本文件
1. 在 **数据连接** 中，拖拽一个 `inject` 节点。
2. 拖拽一个 `read file` 节点，然后双击该节点。
3. 输入文件路径和输出配置。

<img width={300} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/97.png" />

4. 单击 **完成**，然后在末端连接一个 `debug` 节点。
5. 单击右上角 **部署**，接着单击 `inject` 节点触发流程。

<img width={600} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/98.png" />

## 连接Excel文件
1. 在 **数据连接** 中，拖拽一个 `inject` 节点。
2. 拖拽一个 `read file` 节点，然后双击该节点。
3. 输入文件路径和输出配置。

<img width={300} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/99.png" />

4. 单击 **Done**，然后按顺序连接 `book`、`sheet` 和 `sheet to json` 节点，将表格数据转换为JSON格式。

<img width={400} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/100.png" />

5. 单击右上角 **Deploy**，接着单击 `inject` 节点触发流程.

<img width={600} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/101.png" />
