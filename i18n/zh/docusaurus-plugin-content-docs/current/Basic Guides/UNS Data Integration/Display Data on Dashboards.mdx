## 如何在数据看板中展示数据
采集至平台并经过处理的数据，可在数据看板中以图表、表格、热力图等多种格式进行可视化展示，助力全面洞察数据价值。

1. 登录平台，选择 **数据管理** > **数据看板**。

2. 单击 **新建数据看板** 添加一个数据看板。
3. 单击数据看板名称开始编辑。
4. 单击 **Add visualization**，在弹出窗口中为数据看板选择数据源：
    - 默认的 **tmsc** 是存储 **数据建模** 中数据的内置数据库。
    - 单击右下角 **Configure a new data source** 可配置其他数据源（如 MQTT）。
