1. 登录 supOS 系统，然后选择 **数据管理** > **事件流程**。

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/109-1.png" />

<table>
  <thead>
    <tr>
      <th>序号</th>
      <th>项目</th>
      <th>说明</th>
    </tr>
  </thead>
  <tbody>
  <tr>
      <td rowspan="2">1</td>
      <td>搜索</td>
      <td>搜索已存在的数据流。</td>
    </tr>
    <tr>
      <td>新建事件流程</td>
      <td>添加一条新的数据流。</td>
    </tr>
    <tr>
      <td rowspan="1">2</td>
      <td>流列表</td>
      <td>显示数据流信息，单击流名称可开始编辑数据流。</td>
    </tr>
    <tr>
      <td>3</td>
      <td>显示方式</td>
      <td>切换数据流的卡片式/列表式显示。</td>
    </tr>
    <tr>
      <td>4</td>
      <td>流操作</td>
      <td>对数据流进行复制/编辑/删除操作。</td>
    </tr>
  </tbody>

2. 单击 **新建事件流程**，输入名称后单击 **保存**。
3. 单击刚创建的数据流，构建数据流以分析数据。
