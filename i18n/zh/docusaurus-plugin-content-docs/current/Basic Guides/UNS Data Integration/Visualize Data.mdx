## 如何用事件流程构建数据看板
采集至平台并经过处理的数据，可在平台中以自定义样式的图表、表格、文本等格式进行可视化展示。

1. 登录平台，选择 **数据管理（UNS）** > **事件流程（Event Flow）**。
2. 单击 **新建事件流程** 添加一条流。
3. 单击流名称开始编辑。
4. 单击右上角 <img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/111.png" /> 图标，然后单击 **节点管理**。

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/110-1.png" />

5. 在 **安装** 页面搜索并安装 **@flowfuse/node-red-dashboard** 节点。

<img width={450} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/112-1.png" />

6. 使用 flowfuse 的节点构建数据看板：
:::tip
单击右上角 <img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/113-1.png" /> 图标，然后单击 **Dashboard2.0** 可调整看板布局。

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/113.png" />

:::
