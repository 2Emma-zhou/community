## User Journey
Follow the order displayed in the image, you will complete your first data flow in Tier0.

<img width={450} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/48.png" />


- Tier0 builds data models in a folder-file structure, making it easy to track and understand;
- It collects various types of data from diverse sources in **Source Flow**;
- Data collected to Tier0 can be processed in many ways in **Event Flow**;
- Processed data can be displayed on dashboards for better understanding through both **Event Flow** and **Dashboards**.
- Data collected to and processed in Tier0 can be obtained by third parties through <a href="../../../openapi/#/">Open API</a> or MQTT subscription.

:::note
- **Source Flow** and **Event Flow** are based on **NodeRed**. For detailed instructions, please see <a href="https://nodered.org/docs/tutorials/first-flow">NodeRed</a>.
- **Dashboards** is based on **Grafana**. For detailed instructions, please see <a href="https://grafana.com/docs/grafana/latest/dashboards/build-dashboards/create-dashboard/">Grafana</a>.
:::
