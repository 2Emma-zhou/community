## 用户操作流程
按照图片所示顺序操作，即可完成数据流配置。

<img width={450} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/48-1.png" />

- 平台采用**文件夹-文件**结构构建数据模型，便于追踪和理解数据层级；
- 在**数据连接**中采集来自不同数据源的各类数据；
- 采集到平台的数据可在**事件流程**中进行多种方式的处理；
- 处理后的数据流可通过**事件流程**或**数据看板**模块，在可视化面板中展示，更直观地呈现数据价值；
- 采集到平台并经过处理的数据，第三方可通过 <a href="../../../openapi/#/">OpenAPI</a> 或 MQTT 订阅获取。

:::note
- **数据连接**和**事件流程**基于可视化流编程工具开发，详细使用说明请参考 <a href="https://nodered.org/docs/tutorials/first-flow">NodeRed 官方文档</a>；
- **数据看板**模块基于数据可视化平台开发，详细配置指南请参考 <a href="https://grafana.com/docs/grafana/latest/dashboards/build-dashboards/create-dashboard/">Grafana 官方文档</a>。
:::
