- supOS builds data models in a folder-file structure, making it easy to track and understand;
- It collects various types of data from diverse sources in **SourceFlow** module;
- Data collected to supOS can be processed in many ways on **EventFlow** module;
- Processed data can be displayed on dashboards for better understanding through both **EventFlow** and **Dashboards**.
Follow the order displayed in the image to complete the data flow.
