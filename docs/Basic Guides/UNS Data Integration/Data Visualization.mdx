## How to Build a Dashboard
Data collected to, processed on supOS, can be visualized in charts, tables, heat maps and other formats on supOS as well.


1. Log in to supOS, and select **UNS** > **Event Flow**.
2. Click **New Event Flow** to add a flow.
3. Click the flow name to start editing.
4. Click <img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/111.png" /> at the upper-right corner, and then click **Node Management**.

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/110.png" />

5. Under **Install**, search for and install **@flowfuse/node-red-dashboard**.

<img width={450} src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/112.png" />

6. Use nodes under flowfuse to build a dashboard.
:::tip
Click <img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/113-1.png" /> at the upper-right corner, and then click **Dashboard2.0** to adjust the layout.

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/113.png" />

:::
