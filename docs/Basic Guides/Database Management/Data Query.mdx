You can make SQL queries to databases added on **DBConnect**.

1. Log in to supOS-CE, and then select **DevTools** > **SQLEditor**.
2. Select a host and a database on the left side, and then double-click a table to view.

<img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/74.png" />

3. Click <img src="http://communityimage2.oss-cn-hangzhou.aliyuncs.com/75.png" /> next to the table just opened to start SQL query.
